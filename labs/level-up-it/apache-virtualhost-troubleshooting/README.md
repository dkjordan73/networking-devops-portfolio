# Apache Virtual Host Troubleshooting & Remediation

This was a group project that was completed as part of the *Level Up in IT* training program.

The lab environment introduced intentional failures across multiple Apache virtual hosts. The investigation, root cause analysis, automation, and documentation were completed as part of the group coursework.

## What This Project Demonstrates
- Linux filesystem and permission validation
- Apache virtual host troubleshooting
- Safe remediation using Bash automation
- Structured problem-solving under failure conditions
